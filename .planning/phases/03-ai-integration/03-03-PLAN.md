Implementation details:

**Collapsible toggle (D-19):**
- Find the `.panel__header` inside `panelElement`
- Create a toggle button: `<button class="reasoning-toggle" aria-expanded="true" aria-controls="reasoning-log">▼</button>`
- Append toggle button to the panel header
- On click: toggle `panelElement.classList.toggle('panel--collapsed')`, update `aria-expanded`, toggle button text between `▼` and `▶`
- When collapsed, the `#reasoning-log` `<ol>` is hidden via CSS (`max-height: 0; overflow: hidden`)

**AI status badge:**
- Find the `.badge` span inside `panelElement` header
- Store reference as `badgeEl`
- Subscribe to simState: when `state.aiStatus` changes, update badge:
  - `'idle'` → text `'AI Idle'`, class `badge--muted`
  - `'active'` → text `'AI Active'`, class `badge--accent`
  - `'inferring'` → text `'Inferring...'`, class `badge--accent`
  - `'mock'` → text `'Mock Mode'`, class `badge--muted`
  - `'degraded'` → text `'Degraded'`, class `badge--warning`
  - `'off'` → text `'AI Off'`, class `badge--muted`

**Structured reasoning entries:**
- Find `#reasoning-log` `<ol>` element
- Subscribe to simState: when `state.reasoningLog` changes (compare length to last rendered count), render new entries
- Each `ReasoningEntry` renders as:

```html
<li class="reasoning-entry">
  <div class="reasoning-entry__header">
    <span class="reasoning-entry__waypoint">WP #{waypointIndex}</span>
    <span class="reasoning-entry__source reasoning-entry__source--{source}">{source}</span>
    <span class="reasoning-entry__latency">{latencyMs}ms</span>
  </div>
  <div class="reasoning-entry__perception">
    <strong>Perception:</strong>
    {nearbyTrees.length} trees nearby, {coverageStats.percentComplete}% complete, alt {terrain.currentAltitude}m
  </div>
  <div class="reasoning-entry__decision">
    <strong>Decision:</strong>
    {decision ? `${decision.type} (${Math.round(decision.confidence * 100)}% confidence)` : 'No decision (sweep-only)'}
  </div>
  {decision?.reasoning ? `<div class="reasoning-entry__rationale"><strong>Rationale:</strong> ${decision.reasoning}</div>` : ''}
</li>
```

- Scroll to bottom after adding new entries: `logEl.scrollTop = logEl.scrollHeight`
- Auto-expand panel when first AI entry arrives (if collapsed)

**Cleanup:**
- `destroy()` unsubscribes from simState (call the unsubscribe function returned by `subscribe()`)
